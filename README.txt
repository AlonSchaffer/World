How to run WorldMap API:

* Make sure you have the following Modules in python :
  - redis 
  - pika
* Make sure docker and docker-compose installed in your machine.

1. Open your shell, and go in the WorldMapApi directory.
2. Type the command 'docker-compose up' to initiate the docker instances.
3. Make sure the containers are up.
4. run the script 'send-timezones.py'
And you are done!
